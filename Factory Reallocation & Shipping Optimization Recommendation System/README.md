# Factory Reallocation & Shipping Optimization Recommendation System

## Overview

This project focuses on optimizing factory-product assignments and shipping operations for Nassau Candy Distributor. The system analyzes logistics data, evaluates factory reallocations, and generates recommendations to reduce lead times, improve operational efficiency, and support data-driven supply chain decision-making.

The project combines data analytics, machine learning, simulation modeling, and interactive dashboard visualization to identify optimal factory assignments for products.

---

## Business Problem

Traditional factory allocation methods often rely on static business rules and historical practices, resulting in:

- Increased lead times
- Higher logistics costs
- Inefficient factory utilization
- Limited visibility into alternative allocation strategies

The objective is to identify better factory assignments that improve supply chain performance.

---

## Objectives

- Analyze shipping and operational performance
- Identify factory reassignment opportunities
- Simulate alternative factory allocations
- Estimate lead-time improvements
- Measure profitability impact
- Rank recommendations using a scoring framework
- Build an interactive decision-support dashboard

## Methodology

### 1. Data Preprocessing

- Data cleaning

### 2. Factory Reallocation Simulation

The simulation evaluates:

- Alternative factory

### 3. Recommendation Scoring

Recommendations are ranked using:

Generated KPIs include:
